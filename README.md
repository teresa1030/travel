# 高雄旅遊資訊查詢網站
以AJAX抓取開放資料，實作出完整靜態網頁
提供使用者查詢、瀏覽高雄旅遊資訊

## 功能介紹
* 利用JavaScript提供的XMLHttpRequest，抓取高雄市政府開放資料中的JSON格式資料
* 呈現高雄旅遊資訊給使用者
* 設計下拉式選單及按鈕，來瀏覽不同行政區的景點
* 以CSS完成開發，不套用既有模板

## 開發環境
* Html
* CSS
* JavaScript
